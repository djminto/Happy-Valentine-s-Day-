# Happy-Valentine-s-Day-

A Valentine's Day special site with:
- A playful "Will you be my Valentine?" flow
- A moving "No" button interaction
- Heart gift reveal with confetti
- Romantic video + animated letter reveal
- Background music controls

## Run locally

Open `index.html` directly in a browser, or use any static server.

## Deploy to Vercel

1. Go to Vercel Dashboard and click **Add New Project**.
2. Import this repository: `djminto/Happy-Valentine-s-Day-`.
3. Framework preset: **Other** (or leave auto-detected).
4. Build command: **empty**.
5. Output directory: **empty**.
6. Click **Deploy**.

## Important media notes

- Video file must remain exactly at `Image/Mahalia.mp4` (case-sensitive on Vercel).
- If the video codec is unsupported in a browser, re-export as H.264/AAC MP4.
- Background music may require one user tap due to browser autoplay policy.
